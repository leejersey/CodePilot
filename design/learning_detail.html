掌握 Python 协程

协程（Coroutines）是 Python 中实现**单线程并发**的核心机制。与线程不同，协程由开发者控制切换时机，开销极小。

  • 使用 async def 定义异步函数。
  • 使用 await 挂起当前任务，释放控制权。
  • 所有协程必须在 **Event Loop** 中运行。

import asyncio

async def say_hello(n, delay):
    await asyncio.sleep(delay)
    print(f"Task {n} completed after {delay}s")

async def main():
    # 并行运行多个协程
    await asyncio.gather(
        say_hello(1, 1),
        say_hello(2, 0.5)
    )

asyncio.run(main())
