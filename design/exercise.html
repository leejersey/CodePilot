实现异步数据采集器

task_alt 任务目标

  • 1. 实现一个异步函数 fetch_data(api_id: int)。

  • 3. 处理超时异常：如果延迟超过 1.2 秒，应重试最多 3 次。

import asyncio
import random

async def fetch_data(api_id: int):
    """实现你的异步逻辑"""
    retries = 3
    for i in range(retries):
        delay = random.uniform(0.5, 1.5)
        await asyncio.sleep(delay)
        if delay <= 1.2:
            return f"Data from {api_id}"
    raise TimeoutError("请求超时")
